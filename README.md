# Product Preview Card Component

A responsive product card built with semantic HTML and mobile-first CSS.

## Overview

This project is a solution for the Frontend Mentor Product Preview Card challenge.
The layout adapts for mobile and desktop screens and includes hover and focus states
for interactive elements.

## Built With

- HTML5
- CSS3
- CSS custom properties
- Flexbox
- Mobile-first workflow

## Features

- Responsive layout for small and large screens
- Product image switches between mobile and desktop versions
- Accessible button with hover and keyboard focus styles

## Project Structure

- `index.html` - page structure
- `styles.css` - styling and responsive behavior
- `images/` - icons and product images

## Run Locally

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
